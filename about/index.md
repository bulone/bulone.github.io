# 关于

这么几个月，终于把**关于**给补上了。一直没增加的原因是不知道写什么，写自己吧没什么可以写的内容，白纸一张没出出彩的地方。网站也没有什么独特的内容不值得去书写，现在加入这位伙伴，使得网站更加充实吧。

网站取名*吐司气泡*，没有什么多大含义，纯粹就是爱吃吐司面包，后缀是为了注册域名用的。当时注册域名名称时也是思考了许久，简单通用英文域名注册太贵，简单的随机创单词也想不出含义。于是就用喜欢的吐司加上气泡注册了这个新人壹圆的Top域名。

此站点文章内容类型不限，想到啥看到啥就敲啥，从宇宙到水底都可以，当然文笔是一般的几乎大白话。目前就能敲这么多，以后有想法在加进来吧，这就结束了😶‍🌫️。
