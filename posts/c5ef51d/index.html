Telnet开启

启用Telnet客户端

基于Windows10，其他版本请自行查询。

  1. win键+s 输入控制面板并点击进入；
  2. 选择 程序与功能；
  3. 点击 启用或关闭Windows功能；
  4. 选择 Telnet客户端 点击勾选；
  5. 等待配置完成后重启生效。

开启路由Telnet

  1. 在您的电脑上下载并安装Python，安装时请勾选 Add Python To Path；

  1. 下载OpenWRTInvasion-0.0.8并进行解压；
  2. 在文件管理器中打开刚才解压的文件，并在地址栏输入 cmd 回车进入Windows命令窗口；

  1. 在命令行中输入
1
2

pip install requests
python remote_command_execution_vulnerability.py
  1. 输入路由器管理地址 192.168.31.1， 然后回车;
  2. 输入路由器管理密码，然后回车；
  3. 输入 1 ，然后回车等待结束。

Breed刷入

上一节已经将路由器Telnet模式开启。

  1. 命令行输入 ；
1

telnet 192.168.31.1
  1. 用户名和密码均为root；
  2. 连接成功后备份相关文件。
1

dd if=/dev/mtd3 of=/tmp/eeprom.bin
  1. 下载WinScp，协议选择 Ftp，用户名和密码均为 root ；
  2. 登陆完成右侧部分找到 tmp 文件夹并将其中的 eeprom.bin 右键下载到电脑上；
  3. 下载对应的 Breed并改名 breed.bin ,拖拽到 tmp 文件夹内；
  4. 命令行中输入
1

mtd write breed.bin
  1. 等待刷写完成。

固件刷入

本固件来自恩山论坛，仅有基础功能，没有其他的附加功能。

  1. 将第三方固件下载到电脑上等待使用；
  2. 首先用网线一端连接路由器的Lan口，另一端连接电脑。电脑不要连接网络；
  3. 将路由器电源断开，按住 reset 键插入电源，持续时间5秒以上松开即可；
  4. 电脑显示连接以太网，打开路由器并在地址栏输入 192.168.1.1；
  5. 点击固件更新，在 EEPROM 一栏选择刚才备份的 eeprom.bin ,然后点击上传；
  6. 点击固件备份，选择编辑器固件然后文件现在下来备份好；
  7. 点击固件更新，在固件一栏选择下载的第三方固件，点击上传，等待更新完成即可；
  8. 刷好估计固件后，在固件的管理后台进行双清操作，避免部分功能无法使用。
